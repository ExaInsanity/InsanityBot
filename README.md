## :heavy_check_mark: Features :heavy_check_mark:
### Moderation
- Warns
- Verbal Warns
- Blacklist
- Whitelist
- Mute (WIP)
- Tempmute (WIP)
- Unmute (WIP)
- Kick
- Ban
- Tempban (WIP)
- Unban (WIP)
- Modlog
- Unwarn
- Lock/unlock channel
- Purge (mass-delete)
<br />

### Utility
- Tags (custom commands)
- Suggestions (staff and community denied)
<br />

### Tickets
- Open/Close
- Add/Remove users
- Report tickets
- Application tickets
<br />

### Experience (WIP)
- Leaderboard
- Rank command
- Level up messages
<br />
<br />
